# Smart Shopping List

Cross-platform shopping list app built with Expo + React Native and Supabase.

## Highlights
- Email/password auth with persisted sessions.
- Lists and items with per-shop layouts and templates.
- AI assisted layout sorting via a Supabase edge function.
- Light/dark mode with a Home header toggle and persisted preference.

## Quick start
1. `npm install`
2. `cp .env.example .env` and set `EXPO_PUBLIC_SUPABASE_URL` and `EXPO_PUBLIC_SUPABASE_ANON_KEY`
3. `npm start`
